This sample application shows how to use the DigitalRune Animation library for 
character animation.

----------------------------------------------
Requirements:
----------------------------------------------
- Microsoft .NET Framework 4.0
- Microsoft XNA GameStudio 4.0

----------------------------------------------
Description:
----------------------------------------------
This sample application is a collection of several animation examples. 
Please read the comments in the source code for more information.
It is recommened to work through the source code of the samples in the same 
order as they are shown in the application.

Press F1 to display the help text at runtime.
