This sample shows how to use DigitalRune Animation to animate Xbox LIVE Avatars.

----------------------------------------------
Requirements:
----------------------------------------------
- Microsoft .NET Framework 4.0
- Microsoft XNA GameStudio 4.0

----------------------------------------------
Description:
----------------------------------------------
This sample application is a collection of several animation examples. Please read 
the comments in the source code for more information. 
It is recommended to work through the source code of the samples in the same order 
as they are shown in the application.

Press F1 to display the help text at runtime.
